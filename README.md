# 🍅 番茄钟 Pomodoro

一个美观实用的番茄钟计时器 Web 应用，采用深色主题设计，支持多种专注模式与自定义设置。

## ✨ 功能特性

- **三种计时模式**
  - 🍅 专注模式（默认25分钟）
  - ☕ 短休息（默认5分钟）
  - 🌙 长休息（默认15分钟）

- **核心功能**
  - 环形进度指示器，直观显示剩余时间
  - 开始 / 暂停 / 重置控制
  - 番茄计数与统计（今日番茄、专注分钟、完成轮次）
  - 计时完成自动播放提示音
  - 自定义时长设置

- **UI/UX**
  - 深色主题配暖色调番茄红配色
  - 毛玻璃卡片效果
  - 平滑动画与微交互
  - 响应式布局，适配移动端

## 🚀 使用方法

直接用浏览器打开 `pomodoro.html` 即可使用，无需安装任何依赖。

## 🛠️ 技术栈

- React 18
- Tailwind CSS
- 原生 JavaScript
